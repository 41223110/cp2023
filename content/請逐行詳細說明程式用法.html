請逐行詳細說明程式用法

int main(void) {

這裡是 main 函數的開始。int 表示 main 函數會返回一個整數值，而 void 表示 main 函數不接受任何參數。

    // 設定迴圈執行10次
    for (int i = 0; i < 10; i++) {

這行設定了一個 for 迴圈，int i = 0 初始化一個整數變數 i，i < 10 是迴圈的條件，只有在 i 小於 10 時才會執行迴圈中的程式碼，i++ 會在每次迴圈結束時將 i 遞增 1。

        // 在每次迴圈中列印相同字串與 i 變數的值
        printf("Hello World中文 - Line %d\n", i + 1);

這行使用 printf 函數在每次迴圈中輸出一個帶有格式的字串。%d 是一個格式控制符，表示將用整數替換它。i + 1 的值將替換 %d，所以每次迴圈會印出不同的行數。

    }

迴圈的結尾。

    return 0;
}

這是 main 函數的結尾，return 0; 表示 main 函數返回整數值 0，通常表示程序成功結束。這個值會傳遞給操作系統，作為程序的退出狀態碼。
