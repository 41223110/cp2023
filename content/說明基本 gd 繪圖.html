說明基本 gd 繪圖

#include <stdio.h> :寫檔案用的 。

#include <gd.h>:用來創建、著色。

#include <math.h>:數學運算的函數，例如三角函數。

void ： 主要用於表示某個地方不返回任何值，包括函數的返回類型。函數宣告:

void draw_roc_flag(gdImagePtr img);
void draw_white_sun(gdImagePtr img, int x, int y, int size, int color);


int main() {
}

int： 這表示 main 函數將返回一個整數值。這個整數值通常用於指示程序的執行狀態，一般來說，返回 0 表示程序正常結束，而其他的返回值可能表示程序發生了某種錯誤。被用来宣告整數變量。

main： 這是C++程序的主函數。程序的執行從這個函數開始。所有的代碼都在 main 函數內執行。

int：被用来宣告整數變量，例:

int age; // 宣告一個整數變量 age

age = 25; // 將整數值 25 赋给 age 變量

或

int age = 25; // 宣告並初始化一个整數變量 age，初始值為 25
